## setting up

grab the latest beta `.mrpack` file from GitHub releases and import it into PrismLauncher

**JVM tuning recomendations**

playing this modpack on regular PrismLauncher defaults for Forge 1.20.1 will cause stutters and JVM garbage collection to freeze the game when playing. **Change the default minimum JVM memory allocation to `-Xms:6090M` (6GB) and maximum to `-Xmx:6090M` (6GB) for a good experience**. 

this modpack has been tested with Java 21 (Eclipse Adoptium) and will run fine, although the default recomendation of Java 17 will also run fine if you want maximum stability
